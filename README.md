# 🏠 Imobiliária Santos - Projeto Next.js

Este é um projeto de uma landing page para uma imobiliária, desenvolvido como parte de uma atividade acadêmica. O objetivo foi converter um layout HTML/CSS estático para o framework **Next.js**, praticando conceitos de componentes, estilização global e estrutura de arquivos.

## 🚀 Tecnologias Utilizadas
* [Next.js](https://nextjs.org/) - Versão 16
* [React](https://reactjs.org/) - Versão 19
* CSS3 (Estilização Moderna e Responsiva)
* JavaScript (ES6+)
